Просто

Минималистичный интерфейс, который не мешает пользоваться сервисом.

Нормальная база

Ссылки сохраняются в Postgres и не пропадают после перезапуска приложения.

Что умеет сервис

  • GET /health — проверка жизни сервиса
  • POST /shorten — создать короткую ссылку
  • GET /links — список всех ссылок в JSON
  • GET /links-page — HTML-страница со списком ссылок
  • GET /{code} — редирект по короткому коду
